# Before Launch - Replace These Placeholders

## In index.html

1. **Site URL** (2 places) - Search for `REPLACE_WITH_YOUR_URL`
   - `<meta property="og:url" content="https://REPLACE_WITH_YOUR_URL">`
   - `<link rel="canonical" href="https://REPLACE_WITH_YOUR_URL">`

2. **Twitter/X Profile** (1 place) - Search for `REPLACE_WITH_TWITTER_URL`
   - Footer social link

3. **Instagram Profile** (1 place) - Search for `REPLACE_WITH_INSTAGRAM_URL`
   - Footer social link
